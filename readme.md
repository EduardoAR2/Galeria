# ImageHive
ImageHive es una aplicación web que permite buscar y visualizar imágenes de diferentes categorías, con una interfaz amigable y fácil de usar.

## Tecnologías Utilizadas
- HTML
- CSS
- JavaScript
- Bootstrap icons
- API de Pexels
